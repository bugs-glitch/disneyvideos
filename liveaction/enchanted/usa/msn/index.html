Buddy Icons

  1. Right-click on the image and choose "Save Picture As..." and save the file to your desktop.
  2. Once you finish saving the icon from the pop-up, start up AOL Instant Messenger and on the "My AIM" menu select "Edit Options" which will open up another menu.
  3. On this menu, select "Edit Preferences...".
  4. This will open a pop-up window. Select "Buddy Icons" from the left hand side (the menu window) to load this feature.
  5. From this window,, choose "Browse PC" to go to the location of the image file, and select the image that you just saved to use for your buddy icon. Click OK.
